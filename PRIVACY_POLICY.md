# Privacy Policy

## EDS Site Inspector Privacy Policy

Last Updated: 2024

### Data Collection and Use

EDS Site Inspector (the "Extension") does not collect, store, or transmit any personal information or data from users.

### Permissions

This extension uses the following permissions:

- **`scripting`**: Used to inject content scripts for page analysis
- **`activeTab`**: Used to analyze content of the currently active tab
- **`storage`**: Used to locally store user overlay preferences (show/hide states)
- **`tabs`**: Used for tab management
- **`<all_urls>`**: Used to analyze any website

### Data Storage

This extension only stores user overlay preferences (visibility states of sections, blocks, and default content) in the browser's local storage. This data:

- Is stored only on the user's device
- Is never transmitted to external servers
- Is never shared with third parties

### External Services

This extension does not communicate with external services or servers. All processing is performed locally within the user's browser.

### Policy Changes

If we make changes to this privacy policy, we will notify users on this page.

### Contact

If you have any questions or concerns about privacy, please contact us through the GitHub repository's Issues section.
